# miniapp-ui

[![NPM][npm-version-image]][npm-version-url] [![NPM Downloads][npm-downloads-image]][npm-downloads-url]

一款基于 `Taro` 框架开发的ui组件

```bash
npm install -g @tarojs/cli
```

然后在项目中安装ui

```bash
npm install miniapp_ui
```
